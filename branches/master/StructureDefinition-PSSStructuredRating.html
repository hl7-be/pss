Extension: PSS Rating Extension

Official URL: https://www.ehealth.fgov.be/standards/fhir/pss/StructureDefinition/PSSStructuredRating Version: 1.0.0
Computable Name: PSSStructuredRating
Other Identifiers: OID:2.16.840.1.113883.2.51.22.2.42.16

A clinician-friendly rating, or score, for the recommendation; patient-friendly if the recommendation is patient-facing.

Context of Use

    This extension may be used on the following element(s)

    • Element ID: Resource
    • Element ID: RequestGroup.action

Usage info

Usages:

  • Examples for this Extension: Bundle/getrecommendations-a-response, Bundle/getrecommendations-r-response, RequestGroup/getrecommendations-a-response-group and RequestGroup/getrecommendations-r-response-1

This structure is derived from Extension

Summary

Complex Extension: A clinician-friendly rating, or score, for the recommendation; patient-friendly if the recommendation is patient-facing.

  • ratingType: code: A BodyStructure resource representing the body structure treated, for example, Chest Wall Lymph Nodes.
  • ratingValue: Quantity, Ratio: The total amount of radiation delivered to this volume within the scope of this dose delivery.
  • ratingText: markdown: The number of fractions delivered to this volume.
